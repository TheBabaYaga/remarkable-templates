# reMarkable Templates

A public collection of templates for use with [remarkable-template-manager](https://github.com/TheBabaYaga/remarkable-template-manager), a desktop app for managing templates on your reMarkable device.

## Getting Started

### Design in Figma

Use this Figma file as a starting point for creating or editing templates:

**[Remarkable Templates (Figma)](https://www.figma.com/design/zFCgryzevZXomUjC7ClCoP/Remarkable-Templates?node-id=0-1&t=oBWgloozt8RyA0LU-1)**

Export your frames as SVG or PNG at the correct dimensions (see below), then upload them to your device via the template manager app.

## Template Specifications

Templates must be **PNG or SVG** with these dimensions:

| Device           | Dimensions    |
|------------------|---------------|
| reMarkable 1 & 2 | 1404 × 1872 px |
| reMarkable Pro   | 1620 × 2160 px |

**Filename rules:** Use only letters (a–z, A–Z), numbers (0–9), hyphens (`-`), and underscores (`_`). No spaces or special characters.

## Repository Structure

- **`reMarkable 1 & 2/`** — Templates sized for reMarkable 1 and reMarkable 2 (1404 × 1872).
- **`reMarkable Pro/`** — Templates sized for reMarkable Pro (1620 × 2160).

Each folder contains the same template set at the correct resolution for that device.
